# OpenGL实验项目总览

## 项目简介
本项目包含多个基于OpenGL的图形学实验，涵盖了从2D变换到3D建模、纹理映射、贝塞尔曲线/曲面以及片段测试等多种图形学技术。每个实验都有独立的目录和详细的README文档，方便学习和参考。

## 目录结构

```
opengl_experiments/
├── exp1_2d_transform/     # 2D变换实验
├── exp2_3d_basics/        # 3D基础实验
├── exp3_bezier/           # 贝塞尔曲线和曲面实验
├── exp4_texture_mapping/  # 纹理映射实验
├── final_earth/           # 地球仪绘制实验
└── tutorials/             # OpenGL教程（仅保留lesson12片段测试）
```

## 各个实验项目介绍

### 1. exp1_2d_transform - 2D变换实验
- **实验目的**：学习OpenGL中的2D变换，包括平移、旋转、缩放等基本变换
- **主要内容**：实现了2D图形的变换效果，展示了OpenGL变换矩阵的应用
- **交互功能**：键盘控制变换参数，实时观察变换效果
- **详细说明**：请参考 `exp1_2d_transform/README.md`

### 2. exp2_3d_basics - 3D基础实验
- **实验目的**：学习OpenGL中的3D建模和渲染基础
- **主要内容**：实现了简单的3D模型绘制，包括立方体、球体等基本几何体
- **交互功能**：键盘控制视角变换，观察3D模型的不同角度
- **详细说明**：请参考 `exp2_3d_basics/README.md`

### 3. exp3_bezier - 贝塞尔曲线和曲面实验
- **实验目的**：学习贝塞尔曲线和曲面的原理和实现
- **主要内容**：
  - 实现了交互式贝塞尔曲线绘制
  - 实现了贝塞尔曲面绘制
  - 支持鼠标和键盘交互
- **交互功能**：
  - 贝塞尔曲线：鼠标拖动控制点调整曲线形状
  - 贝塞尔曲面：键盘控制控制点位置，调整曲面形状
- **详细说明**：请参考 `exp3_bezier/README.md`

### 4. exp4_texture_mapping - 纹理映射实验
- **实验目的**：学习OpenGL中的纹理映射技术
- **主要内容**：实现了一个带纹理映射的旋转立方体
- **交互功能**：自动旋转，ESC键退出
- **技术特点**：支持纹理加载，当纹理文件缺失时自动生成默认纹理
- **详细说明**：请参考 `exp4_texture_mapping/README.md`

### 5. final_earth - 地球仪绘制实验
- **实验目的**：综合应用OpenGL技术，实现一个完整的地球仪模型
- **主要内容**：
  - 3D球体建模
  - 纹理映射
  - 光照效果
  - 自动旋转动画
- **交互功能**：
  - 鼠标左键拖动：旋转地球仪
  - 键盘按键w/s：放大/缩小地球仪
  - ESC键：退出程序
- **详细说明**：请参考 `final_earth/README.md`

### 6. tutorials - OpenGL教程
- **实验目的**：学习OpenGL的高级特性
- **主要内容**：仅保留了lesson12片段测试，展示了剪裁测试和Alpha测试两种片段测试技术
- **交互功能**：
  - 按键1：切换到剪裁测试模式
  - 按键2：切换到Alpha测试模式
  - ESC键：退出程序
- **详细说明**：请参考 `tutorials/README.md`

## 环境要求

### 硬件要求
- 支持OpenGL的显卡
- 至少1GB内存

### 软件要求
- Windows操作系统
- GCC/G++编译器（推荐MinGW）
- GLUT/FreeGLUT库
- OpenGL 1.0或更高版本

## 编译和运行指南

### 通用编译命令
每个实验目录下都可以使用类似的编译命令：

```bash
g++ -o <可执行文件名> <源文件名> -lopengl32 -lglu32 -lfreeglut -L. -Iinclude
```

### 运行方式
编译成功后，在实验目录下执行生成的可执行文件：

```bash
./<可执行文件名>.exe
```

### 注意事项
1. 确保每个实验目录下都有必要的依赖文件，包括：
   - `include/GL/glut.h`：GLUT库头文件
   - `freeglut.dll`：FreeGLUT动态链接库
   - `glut32.dll`：GLUT兼容库

2. 部分实验可能需要特定的纹理文件，请确保纹理文件与可执行文件在同一目录下

3. 编译某些文件时可能需要添加`-fpermissive`选项来处理一些类型兼容性问题

## 项目特点

1. **模块化设计**：每个实验独立成目录，便于学习和管理
2. **详细文档**：每个实验都有详细的README文档，解释实验目的、实现原理和使用方法
3. **交互性强**：大多数实验都支持键盘或鼠标交互，便于观察不同参数下的效果
4. **代码注释**：源代码包含详细注释，便于理解和学习
5. **兼容性好**：使用标准OpenGL和GLUT库，支持多种编译器和操作系统

## 学习建议

1. 按照实验顺序学习，从2D变换开始，逐步深入到3D建模和高级特性
2. 仔细阅读每个实验的README文档，理解实验目的和实现原理
3. 尝试修改源代码，调整参数，观察不同效果
4. 结合理论知识，深入理解OpenGL的渲染管线和图形学原理
5. 参考官方文档和其他资源，扩展知识面

## 注意事项

1. 本项目使用的是32位OpenGL库，在64位系统上可能需要特殊处理
2. 部分实验可能存在性能问题，特别是在旧显卡上
3. 请确保操作系统已安装必要的OpenGL驱动
4. 编译和运行过程中遇到问题，请参考各实验目录下的README文档或搜索引擎
